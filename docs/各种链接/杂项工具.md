# 杂项工具

## CLI下的进度条显示
- https://github.com/tqdm/tqdm
除开在python中import，tqdm也可以在普通sh命令行中调用，例如：
```sh
$ find . -name '*.py' -type f -exec cat \{} \; |
    tqdm --unit loc --unit_scale --total 857366 >> /dev/null
100%|█████████████████████████████████| 857K/857K [00:04<00:00, 246Kloc/s]
```
